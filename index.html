Understanding the CSS Box Model

Content Box

This is the content area. We'll give it some padding, a border, and a margin to demonstrate the box model in action.

Padding

The space between the content and the border is called padding. It's the cushion inside the box.

Border

This is the border that separates the padding from the margin. It can be styled with different colors and widths.

Margin

The space outside the border is the margin. It creates a gap between this box and other elements on the page.
